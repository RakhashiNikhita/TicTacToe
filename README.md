
# ❌⭕ Tic Tac Toe Game
A classic Tic Tac Toe game built with HTML, CSS, and JavaScript. This is a simple two-player game that can be played on the same device, designed with a clean user interface and smooth game interactions.

## 🚀 Features
- Two-player mode on the same device.
- Interactive and responsive design.
- Real-time game status updates.
- Win/draw detection and reset functionality.
## 🌐 Live Demo
- https://rakhashinikhita.github.io/TicTacToe/

## 🛠️ Built With
- HTML – Markup language for the structure
- CSS – Styling and layout
- JavaScript – Game logic, turn handling, win conditions

## 🧠 How to Play
- Player 1 : ❌
- Player 2 : ⭕
- Players take turns clicking on empty squares.
- The first to align three same symbols (horizontally, vertically, or diagonally) wins.
- If all squares are filled with no winner, it's a draw.
- Click "New Game" to play again.
